# Conky lua - Linux Mint

#### Install
The script will install acpi, conky and make some links: 
```bash
chmod +x install.sh
./install.sh # default location and resource
```
For custom location
```bash
export location="New_York";./install.sh
```
For custom resource
```bash
export resource="mint";./install.sh
```

#### Usage
```bash
~/.conkystart.sh
```
